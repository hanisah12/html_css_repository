JavaScript Intermediate Questions & Answers

Intermediate Level

  • Q1: What are JavaScript events?
    A: Actions like click, input, or keypress that trigger code.
  • Q2: What is DOM?
    A: Document Object Model, a representation of HTML elements as objects.
  • Q3: What is the difference between null and undefined?
    A: null is intentional empty value, undefined means not assigned.
  • Q4: What are arrow functions?
    A: A shorter way to write functions using =>.
  • Q5: What is a callback function?
    A: A function passed into another function to be executed later.
  • Q6: What is the difference between synchronous and asynchronous code?
    A: Synchronous runs line by line, asynchronous allows tasks to happen in the background.
  • Q7: What is JSON?
    A: JavaScript Object Notation, used for storing and exchanging data.
  • Q8: What is localStorage?
    A: A browser storage system that saves data even after refreshing.
  • Q9: What are template literals?
    A: Strings with backticks that support variables and multi-line text.
  • Q10: What is the difference between forEach and map?
    A: forEach executes for each item, map creates a new array.
