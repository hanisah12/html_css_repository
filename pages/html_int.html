HTML Intermediate Questions & Answers

Intermediate Level

  • Q1: Difference between block and inline elements?
    A: Block takes full width, inline fits content.
  • Q2: What is the difference between id and class?
    A: ID is unique, class can be reused.
  • Q3: What is semantic HTML?
    A: Tags that convey meaning like <header>, <footer>.
  • Q4: What is the purpose of the <form> tag?
    A: To collect user input.
  • Q5: Difference between relative and absolute URLs?
    A: Relative is based on current path, absolute is full link.
  • Q6: What are meta tags?
    A: Provide metadata about the document.
  • Q7: What is the difference between <ol> and <ul>?
    A: Ordered list vs unordered list.
  • Q8: What does the <iframe> tag do?
    A: Embeds another page inside.
  • Q9: What is the use of <span>?
    A: Inline container for styling.
  • Q10: What is the difference between HTML and XHTML?
    A: XHTML is stricter and XML-based.
