CSS Beginner Questions & Answers

Beginner Level

  • Q1: What does CSS stand for?
    A: Cascading Style Sheets.
  • Q2: What is the purpose of CSS?
    A: To style and format HTML elements.
  • Q3: How to link CSS to HTML?
    A: Using the <link rel="stylesheet"> tag.
  • Q4: What are inline styles?
    A: Styles written directly inside an HTML tag.
  • Q5: What is an external CSS?
    A: Styles written in a separate .css file.
  • Q6: What is the difference between class and id selectors?
    A: Class is reusable, id is unique.
  • Q7: How to change text color in CSS?
    A: Using the color property.
  • Q8: How to change background color?
    A: Using the background-color property.
  • Q9: What does the * selector do?
    A: Selects all elements.
  • Q10: How to center text?
    A: Using text-align: center;.
