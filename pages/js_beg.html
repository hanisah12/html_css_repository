JavaScript Beginner Questions & Answers

Beginner Level

  • Q1: What is JavaScript?
    A: A programming language used for web interactivity.
  • Q2: How do you add JavaScript to HTML?
    A: Using the <script> tag.
  • Q3: Difference between let, var, and const?
    A: var is function-scoped, let and const are block-scoped.
  • Q4: How to show an alert in JS?
    A: Using alert("message").
  • Q5: How to write a comment in JS?
    A: // for single line, /* */ for multi-line.
  • Q6: What is an array?
    A: A list of values stored in a variable.
  • Q7: What is a function?
    A: A block of reusable code.
  • Q8: How to get HTML element by ID?
    A: document.getElementById("id").
  • Q9: How to print something in console?
    A: console.log("message").
  • Q10: Difference between == and ===?
    A: == compares values, === compares values and types.
