JavaScript Advanced Questions & Answers

Advanced Level

  • Q1: What is a closure?
    A: A function that remembers variables from its outer scope even after that scope is closed.
  • Q2: What is the difference between prototypal inheritance and classical inheritance?
    A: JavaScript uses prototypal inheritance where objects inherit directly from other objects.
  • Q3: What are promises?
    A: Objects that represent the eventual completion or failure of asynchronous operations.
  • Q4: What is async/await?
    A: Syntax for handling promises in a synchronous style.
  • Q5: What is the difference between deep copy and shallow copy?
    A: Shallow copy copies references, deep copy duplicates full data structures.
  • Q6: What are JavaScript modules?
    A: Files that export and import code using export and import.
  • Q7: What are higher-order functions?
    A: Functions that take other functions as arguments or return them.
  • Q8: What is event delegation?
    A: Using a parent element to handle events of child elements efficiently.
  • Q9: What is the difference between setTimeout and setInterval?
    A: setTimeout runs once after delay, setInterval runs repeatedly.
  • Q10: What are generators?
    A: Special functions using function* that can pause and resume execution with yield.
