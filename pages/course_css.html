CSS

CSS (Cascading Style Sheets) styles web pages by controlling layout, colors, fonts, and spacing. It separates design from HTML, supports responsive design, animations, and advanced effects for attractive, consistent, and user-friendly websites.

Beginner Level

  • CSS defines style for web pages
  • Controls colors, fonts, and backgrounds
  • Selectors target specific elements
  • Box model: margin, border, padding, content
  • Inline, internal, and external CSS
Questions

Intermediate Level

  • Flexbox for layout alignment
  • Grid system for two-dimensional layouts
  • Pseudo-classes and pseudo-elements
  • Media queries for responsiveness
  • Positioning: relative, absolute, fixed, sticky
Questions

Advanced Level

  • CSS variables for reusable styles
  • Transitions and keyframe animations
  • Advanced selectors like :nth-child and :not
  • Gradients, masking, and blend modes
  • Using preprocessors like Sass or LESS
