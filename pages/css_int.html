CSS Intermediate Questions & Answers

Intermediate Level

  • Q1: What is the difference between relative, absolute, and fixed positioning?
    A: Relative shifts from normal position, absolute positions relative to parent, fixed is relative to viewport.
  • Q2: What is the difference between inline-block and block?
    A: Inline-block allows elements to sit inline but still have block properties.
  • Q3: What is z-index?
    A: Controls the stacking order of elements.
  • Q4: What are pseudo-classes?
    A: Special states like :hover, :focus, :nth-child.
  • Q5: What is the difference between em and rem?
    A: em is relative to parent font-size, rem is relative to root font-size.
  • Q6: What is the difference between relative and absolute units?
    A: Relative units depend on parent/root, absolute are fixed values.
  • Q7: What is the difference between inline and external CSS?
    A: Inline is inside HTML tags, external is in a .css file.
  • Q8: What is a CSS grid?
    A: A two-dimensional layout system with rows and columns.
  • Q9: What is Flexbox?
    A: A one-dimensional layout system for aligning items.
  • Q10: What is the difference between relative and fixed width layout?
    A: Relative adjusts with screen, fixed stays constant.
